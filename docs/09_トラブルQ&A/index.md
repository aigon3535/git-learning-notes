# トラブルQ&A


#### コミットをミスった時は？

コミットを訂正できる`reset`を使ってみましょう。
resetにはステージング前に戻す、ステージングも戻すことができます。

```bash
git reset --soft HEAD~1
```

#### ブランチ名を修正したいです。

リネーム機能はないため、新しいブランチを切ることで修正できます。

```bash
git branch -m 新ブランチ名
```

#### コミット履歴を綺麗にしたい。特にマージコマンドが履歴に残って読みづらい。

`revase`というコマンドがあり、コミット履歴を先頭にもってくることができます。
もちろん、マージした時のコミットコメントを運用ルールで統一する解決策もあります。

```bash
git rebase origin/main
```
